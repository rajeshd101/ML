# Spam SMS Detector 📵

## What Does This Do?

This is like a bouncer at a club, but for text messages. It looks at a message and says "Yep, that's spam" or "Nope, that's real."

Think of it like sorting your toys:
- Real toys go in one box
- Broken toys go in another box

This program sorts messages the same way!

## How Does It Work?

1. **You send it a text** → "Hey, you won the lottery!"
2. **It thinks about it** → "Hmm, I've seen 1000 messages like this. This smells like spam."
3. **It tells you** → "This is SPAM (93% sure)"

## The Magic Inside

- **Dataset**: 5,574 real text messages (some spam, some real)
- **Brain Training**: Made it read all these messages and learn the pattern
- **Accuracy**: Gets it right 98 times out of 100

## How to Use It

1. Open the website
2. Type a message
3. Click "Check"
4. See if it's spam or real!

## What It Learned

Spam messages usually have words like:
- "Free"
- "Win"
- "Click here"
- "Urgent"
- "Money"

Real messages usually have:
- Your friend's name
- Normal words
- Personal stuff

## The Numbers That Matter

- **98% Accurate** = Out of 100 messages, it gets 98 right
- **5,574 Messages** = That's how many examples it learned from
- **Real vs Spam** = What it's trying to figure out

---

**Simple Truth**: It's like having a friend who read 5,000+ messages and now knows what spam looks like!
